# Assets Folder

This folder contains static assets for the project including:

## Image Files
- Icons (.png, .svg, .ico)
- Background images
- Logos
- Charts/graphs images for reference

## Other Assets
- Fonts (if using custom fonts)
- Audio files (if needed)
- Video files (if needed)

## File Organization

Please organize files in subfolders:
- `images/` - for all image files
- `icons/` - for icon files
- `fonts/` - for custom font files

## File Naming Convention

Use descriptive names with no spaces:
- Use underscores or hyphens: `group_logo.png` or `group-logo.png`
- Keep names lowercase when possible
- Include dimensions for images when relevant: `icon_32x32.png`
