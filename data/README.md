# Data Folder

This folder will contain all the data files for the visualization project.

Common data formats to be used:
- CSV files (.csv)
- JSON files (.json)
- TSV files (.tsv)

## Data Structure Guidelines

When adding data files, please ensure:
1. Files are properly formatted
2. Column headers are descriptive
3. Data is clean and validated
4. Large files are compressed if necessary

## Example Data Files

Add your data files here following the naming convention:
- `dataset_name.csv`
- `dataset_name.json`

Remember to update this README when adding new data files.
